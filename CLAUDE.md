# Claude Code Orientation

This project uses `AGENTS.md` as the **single source of truth**. Read it first — it is the consolidated constitution.

For visual decisions (color, motion, layout, materials, tokens), also read `docs/visual-direction.md`.

All other docs are loaded on demand only:

- `docs/roadmap.md` — phase tracking, future components, future admin panel
- `docs/decisions/` — architectural decision records
- `specs/NNN-*/` — feature specs (spec-kit, when introduced)

Always run `npm run build` and `npm run lint` before reporting completion.
